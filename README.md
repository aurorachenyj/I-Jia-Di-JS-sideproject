網站功能－前台
◆ 選擇甜點（可多選）或捷運站點後，輕鬆獲取相關店家資訊，也可觀看評價
◆ 提供會員登入、註冊、收藏、評價、個人資料修改功能

網站功能－後台
◆ 會員管理
◆ 評價管理、圖表統整評價資料

使用技能
◆ 使用原生 JavaScript 開發
◆ 畫面切版使用 SCSS 手刻 RWD
◆ 建立 db.json 資料串接 JSON-server 模擬 RESTful API 
◆ 使用 JSON-server-auth 控制使用者權限
◆ 使用 Render 部署
